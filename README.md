## 🚀 Overview
This project is an AI-powered Network Intrusion Detection System that combines Machine Learning and Deep Learning to detect anomalies and potential cyber threats in real-time network traffic.

It simulates a real-world cybersecurity monitoring system with data processing, threat detection, and visualization dashboard.

---

## 🧠 Features
- Real-time network packet capture
- Feature engineering pipeline
- ML-based anomaly detection
- Deep Learning Autoencoder for unknown attacks
- Threat scoring system
- SQLite-based logging system
- Interactive dashboard
- Dataset generation tools
